Nyanko API Documentation

This API provides access to random images based on specific tags and ratings. Below are the available endpoints and usage examples.

Endpoints

GET /api/references/all_tags.json

Returns a JSON object with all available image tags and ratings.

GET https://www.catgirlnexus.xyz/api/v1/pages/{number}/images

Returns a list of 10 images.

Python Library: animegirlapi

The animegirlapi Python library allows you to easily fetch random images using specified filters. Here's how to use it:

Function: animegirlapi.get_random_image()

This function returns random images based on the provided filters. Below are the parameters you can use:

Parameters:
  • number (int): The number of images to return. Max: 100.
  • tag (Union[str, tuple, list]): Tag(s) to filter the images by. Can be a single string or a list/tuple of strings.
  • rating (Union[str, tuple, list]): Rating(s) to filter the images by (e.g., "safe"). Can be a single string or a list/tuple of strings.
  • ignore (bool): If set to True, ignores warnings if fewer images than requested are available.
Example Usage:
import animegirlapi

# Fetch 5 random images with the tag 'yuri' and rating 'safe'
images = animegirlapi.get_random_image(number=5, tag="yuri", rating="safe")

# Print the fetched images
print(images)

The function will return a list of URLs of the images that match the specified criteria.
