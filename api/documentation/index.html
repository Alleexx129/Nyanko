Nyanko API Documentation

This API provides access to random images based on specific tags and ratings. Below are the available endpoints and usage examples.

Endpoints

GET /api/references/all_tags.json

Returns a JSON object with all available image tags and ratings.

GET https://www.catgirlnexus.xyz/api/v1/pages/{number}/images

Returns a list of 10 images.

Python Library: animegirlapi

The animegirlapi Python package is designed for easy interaction with the CatGirlNexus API. It provides functionalities to fetch image details, download images, and access various tags and ratings. Here's how to use it:

Function: get_info(url)

Retrieves information about a specific image from the API.

Parameters:
  • url (str): The URL of the image.
Returns:

A dictionary with:

  • tags: A dictionary of image tags (excluding "rating").
  • rating: The image's rating.

Example:
info = get_info(f"https://www.catgirlnexus.xyz/api/storage/image{id}.jpg")

Function: get_all_tags()

Gets a dictionary of all available tags with their descriptions.

Returns:

A dictionary where:

  • Keys are tag names.
  • Values are descriptions of the tags.

Example:
tags = get_all_tags()

Function: download_image(url, output=None)

Downloads an image from the specified URL and saves it to the given path.

Parameters:
  • url (str): The URL of the image.
  • output (str, optional): The file path where the image will be saved. If not specified, the image is saved with its filename in the current directory.
Returns:

A dictionary with:

  • path: The path where the image was saved.

Example:
download_image("https://example.com/image1.jpg", "images/image1.jpg")

Function: get_all_images()

Fetches and returns a dictionary of all images from the API.

Returns:

A dictionary with all image URLs and their associated data.

Example:
images = get_all_images()

Function: get_all_ratings()

Gets a list of all possible ratings.

Returns:

A list of ratings, such as "suggestive", "nsfw", etc.

Example:
ratings = get_all_ratings()

Function: get_random_image(number=1, tag=None, rating=None, ignore=False, randomize=False)

Fetches a random image that matches the given tags and/or ratings.

Parameters:
  • number (int): Number of images to fetch (up to 100).
  • tag (str, tuple, list, optional): Tags to filter images by.
  • rating (str, tuple, list, optional): Ratings to filter images by.
  • ignore (bool, optional): Whether to ignore warnings if not enough images are found.
  • randomize (bool, optional): Whether to shuffle images before selection.
Returns:

A list of image URLs matching the specified criteria.

Example:
import animegirlapi

# Fetch 5 random images with the tag 'yuri' and rating 'safe'
images = animegirlapi.get_random_image(number=5, tag="yuri", rating="safe", randomize=True)

# Print the fetched images
print(images)

Requirements

  • Python 3.6 or higher
  • requests library

JavaScript Module: animegirlapi

The animegirlapi JavaScript module provides functions to fetch random images, download images, and retrieve metadata from the API. Below are the available functions:

Function: getRandomImage({ number, tag, rating, ignore, random })

Fetches a random image from the API that matches the specified tag(s) and/or rating(s).

Parameters:
  • number (number): The number of images to request. Default is 1. Must be 100 or fewer.
  • tag (string | string[] | null, optional): A tag or a list of tags to filter images. If null, no tag filtering is applied.
  • rating (string | string[] | null, optional): A rating or a list of ratings to filter images. If null, no rating filtering is applied.
  • ignore (boolean, optional): If false and not enough images are found, a warning is printed. If true, no warning is printed. Default is false.
  • random (boolean, optional): If true, shuffles the images randomly before filtering.
Returns:

Promise: A promise that resolves to a list of URLs for the requested number of images. Returns an empty array if no images are found.

Example Usage:
const { getRandomImage } = require('animegirlapi');

// Fetch 5 random images with tag "yuri" and rating "safe"
getRandomImage({ number: 5, tag: 'yuri', rating: 'safe' })
    .then(images => console.log(images));

// Fetch 10 random images with tag "catgirl" without checking for rating
getRandomImage({ number: 10, tag: 'catgirl' })
    .then(images => console.log(images));

// Fetch 3 random images with rating "explicit" and ignore the warning if not enough images are found
getRandomImage({ number: 3, rating: 'nsfw', ignore: true })
    .then(images => console.log(images));

// Fetch 4 random images with multiple tags
getRandomImage({ number: 4, tag: ['yuri', 'catgirl'] })
    .then(images => console.log(images));

Function: getInfo(url)

Fetches metadata about a specific image URL, including tags and rating.

Parameters:
  • url (string): The URL of the image to fetch metadata for.
Returns:

Promise<{ tag: string[], rating: string }>: A promise that resolves to an object containing the tags and rating of the specified image.

Example Usage:
const { getInfo } = require('animegirlapi');

// Fetch metadata for a specific image URL
getInfo('https://www.catgirlnexus.xyz/api/storage/image${id}.jpg')
    .then(info => console.log(info));

Function: downloadImage(url, path)

Downloads an image from a URL and saves it to a specified local path.

Parameters:
  • url (string): The URL of the image to download.
  • path (string): The local path where the image should be saved.
Returns:

Promise: A promise that resolves when the download is complete.

Example Usage:
const { downloadImage } = require('animegirlapi');
const fs = require('fs');

// Download an image and save it to the 'images' directory
downloadImage('https://example.com/path/to/image.jpg', 'images/image.jpg')
    .then(() => console.log('Image downloaded successfully'))
    .catch(error => console.error('Error downloading image:', error));
