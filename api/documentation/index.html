Nyanko API Documentation

Overview

This API provides access to random images based on specific tags and ratings. Below are the available endpoints and usage examples.

Endpoints

GET /api/references/all_tags.json

Returns a JSON object with all available image tags and ratings.

GET /random-image

Returns a random image based on the specified tags and ratings.

Parameters:

  • number (int): The number of images to return. Max: 100.
  • tag (string or list): Tag(s) to filter the images by.
  • rating (string or list): Rating(s) to filter the images by (e.g., safe).
  • ignore (bool): Ignore warnings if fewer images than requested are available.

Example Request:

import requests

response = requests.get("https://www.catgirlnexus.xyz/api/references/all_tags.json")
print(response.json())
